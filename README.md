# Simple url scanner

## Usage 

```
python3 scanner.py url-file.txt
```
```
./scanner.py url-file.txt
```

## Url file

Contains a list of urls separated by a newline

```
https://google.com/
http://google.com:80/
https://google.com:443/
http://google.com:8080/
https://github.com/
```

Results are appened to output.txt
